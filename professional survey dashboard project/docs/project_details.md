# Project Details

## Objective
To analyze professional survey responses and visualize key trends such as:
- Job roles
- Salary distribution
- Programming languages and tools
- Work-life balance and satisfaction

## Dashboard Features
- Interactive filters and slicers
- KPI cards for quick insights
- Charts for comparison and distribution analysis

## Dataset
The dataset should be placed inside the `data/` folder.
Supported formats:
- CSV
- Excel (.xlsx)

## Notes
If the PBIX file was originally connected to an external path,
re-map the dataset to the local `data/` folder when opening the project.
