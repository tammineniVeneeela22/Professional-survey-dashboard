# Professional Survey Dashboard (Power BI)

## Overview
This project contains a Power BI dashboard built using professional survey data.
The dashboard provides insights into roles, salaries, skills, and demographics.

## Folder Structure
- `Professional survey dashboard.pbix` – Main Power BI report file  
- `data/` – Dataset used for the dashboard (add CSV/Excel files here)  
- `docs/` – Project documentation and details  

## Requirements
- Power BI Desktop (latest version recommended)

## How to Use
1. Open `Professional survey dashboard.pbix` in Power BI Desktop.
2. If prompted, update the data source path to files inside the `data/` folder.
3. Refresh the data to view the dashboard.
